# 签到与邀请

这是两个可选的余额获取途径，是否开启由管理员决定。

## 每日签到

如果平台开启了签到，你可以每天领取一次奖励，奖励直接进入余额。

### 奖励模式

签到奖励可由管理员配置为以下模式（可叠加）：

- **每日固定奖励**：每天签到获得固定金额；
- **连续签到（Streak）奖励**：连续签到天数越多，按周期（如 7 天为一轮）发放阶梯奖励，中断后连续天数重置；
- **随机奖励**：在管理员设定的最小值与最大值之间随机发放。

### 签到规则

- 每天最多签到一次，按管理员配置的**时区**（默认 `Asia/Shanghai`）划分「一天」；
- 「签到状态」会显示今日是否可签、当前连续天数、今日奖励预览，以及下一档连续奖励；
- 「签到记录」可查看历史签到与每次获得的奖励。

在签到页点击签到按钮即可领取。已签到的当天再次请求会被拒绝。

## 邀请返佣

如果平台开启了邀请功能，你会有一个**专属邀请码 / 邀请链接**。

### 工作方式

1. 把你的邀请链接分享给新用户；
2. 新用户通过链接注册后，成为你的「被邀请人」；
3. 被邀请人消费时，平台按设定的**返佣比例**把一部分金额返还到你的余额；
4. 每一笔返佣都会记录在返佣明细中。

返佣比例由平台统一设置（管理员侧 `referral_commission_rate`）。你可以在「邀请」相关页面查看自己的邀请码、已邀请人数与累计返佣。

::: tip
邀请关系在被邀请人**首次注册**时确定，之后不可更改。返佣是按被邀请人的实际消费持续结算的。
:::

管理员侧的配置说明见 [签到与邀请返佣](/admin/checkin-referral)。
