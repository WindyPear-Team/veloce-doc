# 钱包与充值

平台采用**预付费余额**模式：账户里有余额，调用 API 时按用量从余额扣费。

## 查看余额

在 **钱包** 页面可以看到当前余额。余额以平台货币计价，显示符号由管理员配置（默认 `$`）。

## 充值

如果管理员开启了在线支付，你可以在钱包页发起充值：

1. 选择或输入充值金额（平台可能预设了 `5 / 10 / 20 / 50 / 100` 等档位，并设有最低充值额）；
2. 选择支付方式（如支付宝、微信，取决于平台开通的渠道）；
3. 平台按汇率把平台货币金额换算成人民币（默认参考汇率约 `7.20`，以平台实际设置为准）；
4. 跳转到支付网关完成付款；
5. 支付成功后余额自动到账。

支持的支付网关包括易支付（Yipay）与 OpenPayment，由管理员二选一接入。详见管理员侧的 [支付与充值](/admin/payment)。

## 充值订单

在钱包页可以查看历史充值订单及其状态：

| 状态 | 含义 |
| --- | --- |
| `pending` | 已创建，等待支付 |
| `paid` / `success` | 支付成功，余额已入账 |
| `failed` / 过期 | 未完成 |

如果你已付款但余额长时间未到账，请保留订单号联系平台管理员核对。

## 其他获取余额的方式

除了充值，部分平台还提供：

- **每日签到**奖励；
- **邀请返佣**：邀请新用户消费后获得返佣。

详见 [签到与邀请](/guide/checkin-and-referral)。

::: warning 余额不足
当余额不足以支付一次请求时，网关会拒绝该请求。请及时充值或通过签到/返佣补充。
:::
